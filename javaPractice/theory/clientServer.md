


- Client sends request to server - http 
- Server gets the request and responds - this is a response

## Components of a request: 
- method: get, post, put, delete
- url: endpoint
- headers: metadata, auth, content
- Body: data to the server

## Components of a response: 
status code: 200, 404, 500
headers: server, content type, length, cache etc
body: data req by client.


get: Read or retrieve
post: create 
put: update existing data entirely (whole json tree)
patch: partial update (part of json tree)
delete: remove data
